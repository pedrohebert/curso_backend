## 🎨 [cores](cores.py)



Estudo simples sobre o funcionamento de cores no terminal utilizando códigos ANSI e o padrão RGB.

O script gera uma animação contínua variando as cores de fundo a partir de valores `R`, `G` e `B`, criando transições suaves diretamente no shell.

#### 🔧 Conceitos explorados:
- Sequências de escape ANSI (`\033`)
- Cores em modo True Color (RGB)
- Manipulação de saída em CLI
- Animação em loop com `sleep`

> [!NOTE]  
> Requer terminal com suporte a cores RGB.  
> Interrompa com `Ctrl + C`.
